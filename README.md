Rosebank College Magazine Website

This is the official website for Rosebank College Magazine. It showcases the latest updates, creative works, achievements, news, and interviews related to Rosebank College. The website is designed to be visually appealing, user-friendly, and responsive across all devices. It includes sections for the gallery, news, achievements, and inquiries.

Features

Responsive Design: The website adjusts seamlessly across different screen sizes (desktop, tablet, mobile).

Gallery: A modern image gallery showcasing various college-related visuals.

Contact & Inquiries: A form to submit inquiries and feedback.

News & Achievements: Sections dedicated to the latest updates and accomplishments at Rosebank College.

Navigation: Easy-to-use navigation for quick access to all sections of the website.
